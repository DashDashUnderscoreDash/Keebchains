## Detour is a:
- 11u ortho or rowstag layout
- Designed by Nachie
- Many bottom row options
- Knob, 4 extra keys, and an OLED are featured on an angled upper portion of the board
- Leafspring mount or bottom mount
- Can be ran plateless
